# FermentarBK

Bienvenido a **FermentarBK**, un proyecto dedicado a la alquimia de la fermentación funcional y la salud cardiovascular.

### Sobre el proyecto
Este portal, creado por **Blue Kolmer** (autora en Amazon), explora la intersección entre la biotecnología casera y el bienestar arterial. 

### Temas principales:
* **Natto de legumbres españolas:** Adaptando la sabiduría japonesa a nuestro producto local.
* **Vitamina K2 y Limpieza Arterial:** El papel del *Bacillus subtilis* en la salud del corazón.
* **Protocolos de Nutrición:** La importancia de las grasas de cadena media y corta en el transporte de nutrientes.

> [!IMPORTANT]
> **Aviso de seguridad:** El contenido de este sitio es informativo. Debido a la alta potencia biológica de la Vitamina K2, si tienes las arterias bloqueadas o tomas anticoagulantes, consulta siempre con un profesional.
